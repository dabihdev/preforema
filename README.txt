========================= PREtemp FOREcast MAnager (PREFOREMA) =========================

ABOUT
============
PREFOREMA è un'applicazione volta alla velocizzazione della previsione PRETEMP. Nello
specifico PREFOREMA permette di:
1. Selezionare il giorno di previsione (giorno di previsione preimpostato: domani)
2. Creare una nuova cartella di progetto con la data di previsione, e all'interno 3
files:
- 1 file .docx già formattato dove scrivere la previsione
- 1 file .svg contenente il template della mappa PRETEMP
- 1 file .html dove esportare il testo della previsione una volta compilata.
3. Esportare il testo della previsione da Word direttamente sulla pagina HTML che sarà
poi caricata online.

REQUISITI
============
L'app è ancora in fase di test. Dovrebbe funzionare correttamente su sistemi operativi
Windows a 64bit.

INSTALLAZIONE
=================
Basta scaricare la cartella preforema/ nel proprio pc, andare alla cartella app/ e
cliccare sul file eseguibile "preforema.exe" per eseguire l'applicazione.

CONSIGLI UTILI
=================
> PREFOREMA aprirà automaticamente i file generati dal progetto (il .docx e l'.svg) così
da preparare subito l'ambiente di lavoro. Quando esportate il testo sulla pagina HTML,
PREFOREMA aprirà anche la pagina html sul browser.
> PREFOREMA dovrebbe usare le applicazioni preimpostate del vostro sistema operativo
per aprire questi file (per esempio MS Word per il .docx e Inkscape per lo .svg).
> Il file svg contiene la mappa di PRETEMP, le date e i nomi degli autori aggiornati, più
4 quadrati colorati con i livelli di pericolosità. Sul programma di grafica utilizzato
(per esempio Inkscape) sarà possibile copiare lo stile dei quadrati per disegnare aree di
pericolosità aventi già il colore desiderato (ESEMPIO INKSCAPE:
1. Seleziona il quadrato con il colore desiderato.
2. Premi Ctrl+C, poi Ctrl+V.
3. Ora premi lo strumento per disegnare e traccia l'area di pericolosità.
4. L'area disegnata avrà già lo stile (i colori, la trasparenza, ecc.) desiderato per quel livello di pericolosità.
